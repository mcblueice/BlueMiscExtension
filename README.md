
# BlueMiscExtension

一個藍冰伺服器用於擴展雜項功能的插件

## 功能特色

- 減少生物受擊時噴出的粒子效果(damage_indicator)

## 指令列表

| 指令 | 用途 | 權限節點 |
|------|------|------|
| `/bluemiscextension reload` | 重新載入設定檔 | `bluemiscextension.reload` |
